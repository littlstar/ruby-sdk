# Littlstar Ruby Gem

This gem encapsulates the functionality needed to access content through the Littstar API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'littlstar'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install littlstar
```
